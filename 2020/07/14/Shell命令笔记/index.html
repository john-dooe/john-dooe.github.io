Shell命令笔记

Bash

目录

ls

列出当前目录下的内容

tree

以树状图列出当前目录结构，基本相当于 ls -R，但更加直观

pwd

显示当前目录

cd

切换当前目录

find

寻找文件或目录

用 “.” 表示当前目录，用 “/” 表示根目录

mkdir

在当前目录下创建一个新文件夹

文件

rm

cat

连接文件并打印到标准输出设备上

file

通过文件内容推断文件类型

grep

查找文件里符合条件的字符串

hexdump

最简单的看文件 16 进制的方法

strings

打印出文件中所有的 ASCII 字符

  • 压缩

  • 解压缩

文本处理

vim

vim 是从 vi 发展出来的一个文本编辑器

一开始会进入到命令模式，当进入其他模式时按下Esc可以返回命令模式

  • 按下i进入输入模式

    直接输入就行

  • 按下:进入底线命令模式

    输入命令，按下回车即可

    • w：保存

    • q：退出

    • wq：保存并退出

nano

有的时候系统中没有 vim，但还要先给 apt 换源，那就只能用 nano 了

一开始就可以直接输入

  • Alt+U：撤销操作

  • Alt+E：恢复已撤销操作

  • Ctrl+O：保存

  • Ctrl+X：退出
